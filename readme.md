# 说明

`service`为客户端、web提供接口；调度深度学习；另外提供其他的服务。

## 目录说明

### 项目跟踪目录

- `doc`为文档
- `src`为项目业务代码路径
- `hooks`为项目钩子脚本，限制提交
- `pack`为项目自动化构建
- `third_party`为`submodule`第三方库
- `test`为测试代码

### 开发时目录

- `bin`用于存放第三方程序
- `include`用于存放第三方头文件
- `lib`用于存放第三方库
