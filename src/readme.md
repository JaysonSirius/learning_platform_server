# 说明

- `base_fun`包含一些基本的模块:
  - `sdk`网络及协议
  - `libuv`实例
  - `任务管理`
- `local_fun`包含项目实现的业务功能模块
  - 包含深度学习调度
  - 告警
  - 会话管理
  - 升级管理
  - 用户管理
- `mkconfig`包含项目编译规则
- `sys_ctrl`包含系统配置相关
  - 版本信息
  - 软件基本设置：`coredump`配置，日志配置等
  - 系统能力集、只读配置、运行时参数
- `third_fun`包含对接第三方平台和调用第三方业务功能的模块
- `tools`包含工具函数、宏、功能等
  - `cordump`启用
  - 简单打印
  - `crc16`
- `usr_apps`用于模块管理
